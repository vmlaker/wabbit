Wabbit is a webcam snapshot recorder and archive server. It comes with a web API and a live surveillance page.

The recorder is written in C++ as a shared-memory, multi-threaded image processing pipeline. The pipeline implements snapshot capture, disk storage and database indexing stages. It is designed to be easily extended with additional image processing steps if needed.

Web server and database tools are written in Python. The web API is implemented in the Flask microframework, and allows random access lookup of stored images. A pruner process maintains the image archive to a configured maximum length of time.

Wabbit is written by Velimir Mlaker using open source technologies:

  • SCons for C++ build
  • OpenCV for camera access
  • ODB for C++ ORM
  • SQLAlchemy for Python ORM
  • MySQL for RDBMS
  • Flask for web app
  • Jade and CoffeeScript for the client
  • jQuery for AJAX
